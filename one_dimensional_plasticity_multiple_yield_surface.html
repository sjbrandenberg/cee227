Elasto-Plastic Multiple Yield Surface Plasticity Model#

Define Model#

We have N number of yield surfaces, each with their own kinematic hardening modulus.

i. Elastic stress-strain relationship#

\(\dot{\sigma} = E\left(\dot{\epsilon} - \sum_{k=1}^{N}{\dot{\epsilon}^{p}_k}\right)\)

ii. Hardening variables#

\(\sigma_{y,k}\) = Array of yield stresses

\(\sigma_{q,k}\) = Array of back stresses

\(H_k\) = Array of kinematic hardening moduli

\(k\) = index counter over number of yield surfaces

iii. Yield Surfaces#

\(f_k(\sigma,\sigma_{y,k}, \sigma_{q,k}) = |\sigma - \sigma_{q,k}| - \sigma_{y,k}\)

iv. Flow rule#

\(\dot{\epsilon}^p_k = \gamma_k \cdot sign\left(\sigma - \sigma_{q,k}\right)\)

v. Hardening Law#

\(\dot{\sigma_{y,k}} = 0\)

\(\dot{\sigma_{q,k}} = H_k \cdot \dot{\epsilon}^p_k\)

\(\dot{H_k} = 0\)

vi. Kuhn-Tucker complementary conditions#

\(\gamma_k \geq 0\)

\(f\left(\sigma, \sigma_{y,k}, \sigma_{q,k}\right) \leq 0\)

\(\gamma_k \cdot f\left(\sigma, \sigma_{y,k}, \sigma_{q,k}\right)=0\)

vii. Consistency condition#

\(\gamma \cdot \dot{f_k}\left(\sigma, \sigma_{y,k}, \sigma_{q,k}\right) = 0\)

Return Mapping#

\(\sigma_{i}\) = stress value at start of current load step

\(\Delta \epsilon = \epsilon_{i+1} - \epsilon_i\) = strain increment

\(\sigma_{i+1}^{trial} = \sigma_{i} + E \cdot \Delta \epsilon\)

\(\sigma_{q,i+1}^{trial} = \sigma_{q,i}\)

\(f_{i+1}^{trial} = \left|\sigma_{i+1}^{trial} - \sigma_{q,i+1}^{trial}\right| - \sigma_y\)

if \(f_{i+1}^{trial} < 0\):

\(\sigma_{i+1} = \sigma_{i+1}^{trial}\)

\(\epsilon^p_{i+1} = \epsilon^p_i\)

\(\sigma_{q,i+1} = \sigma_{q,i+1}^{trial}\)

else:

\(\Delta \gamma = Mat^{-1}\cdot vec\)

\(Mat_{kl} = E \cdot sign(\dot{\epsilon}) + H_k\cdot sign(\dot{\epsilon})\) if \(k=l\)

\(Mat_{kl} = E \cdot sign(\dot{\epsilon})\) if \(k \neq l\)

\(vec_{k} = E \cdot \dot{\epsilon} + \sigma_{n} - \sigma_{q_k} - \sigma_{y_k}\)

\(\sigma_{n+1} = \sigma_{n} + E \left(\dot{\epsilon} - \sum_{k=1}^N{\Delta \gamma_k \cdot sign(\dot{\epsilon}} )\right)\)

%matplotlib notebook
import numpy as np
import matplotlib.pyplot as plt

E = 100000
N_surf = 30
#H = np.asarray([10.0, 1.0, 0.3, 0.0])*E
H = np.linspace(2.0, 2.0, N_surf)*E
sigmay = np.linspace(50, 2000, N_surf)
epsilon_y = sigmay[-1]/E
Npts = 500 # Number of points per cycle in sinusoidal strain history
Ncyc = 2 # Number of cycles in sinusoidal strain history
N = Npts*Ncyc
time = np.arange(0,N)
epsilon_max = 20.0*epsilon_y
epsilon = []
for i in range(N):
    epsilon.append(epsilon_max*np.sin(2.0*np.pi*i/Npts))

# Now define stress
sigma = np.zeros(N)
sigma_q = np.zeros((N_surf,N))
epsilon_p = np.zeros((N_surf,N))
ftrial = np.zeros(N_surf)
Mat = np.full((N_surf,N_surf),E) + np.diag(H)
for i in range(1,N):
    depsilon = epsilon[i] - epsilon[i-1]
    if(depsilon==0.0):
        for j in range(N_surf):
            sigma_q[j,i] = sigma_q[j,i-1]
            epsilon_p[j,i] = epsilon_p[j,i-1]
        sigma[i] = sigma[i-1]
        continue
    elif(depsilon>0):
        SIGN = 1.0
    else:
        SIGN = -1.0
    sigma_trial = sigma[i-1] + E*depsilon
    Nyield = 0
    for j in range(N_surf):
        sigma_qtrial = sigma_q[j,i-1]
        ftrial[j] = np.abs(sigma_trial-sigma_qtrial) - sigmay[j]
        if(ftrial[j]<=0):
            sigma_q[j,i] = sigma_qtrial
            epsilon_p[j,i] = epsilon_p[j,i-1]
        else:
            Nyield += 1
    if(Nyield == 0):
        sigma[i] = sigma[i-1] + E*(epsilon[i] - epsilon[i-1])
    else:
        dgamma = np.linalg.solve(Mat[:Nyield,:Nyield],ftrial[:Nyield])
        while(np.min(dgamma)<0):
            Nyield -= 1
            sigma_q[Nyield,i] = sigma_q[Nyield,i-1]
            epsilon_p[Nyield,i] = epsilon_p[Nyield,i-1]
            dgamma = np.linalg.solve(Mat[:Nyield,:Nyield],ftrial[:Nyield])
            
        sum_depsilon_p = 0.0
        for j in range(Nyield):
            sigma_q[j,i] = sigma_q[j,i-1] + H[j]*dgamma[j]*SIGN
            epsilon_p[j,i] = epsilon_p[j,i-1] + dgamma[j]*SIGN
            sum_depsilon_p += dgamma[j]*SIGN
        sigma[i] = sigma[i-1] + E*(depsilon - sum_depsilon_p)

            
fig, ax = plt.subplots()
ax.plot(epsilon,sigma)
ax.set_ylabel(r'$\sigma$')
ax.set_xlabel(r'$\epsilon$')
ax.grid(True, alpha=0.5)
plt.tight_layout()
