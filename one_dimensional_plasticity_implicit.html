Elastic Perfectly-Plastic - Implicit Integration#

Define Model#

i. Elastic stress-strain relationship#

\(\dot{\sigma} = E\left(\dot{\varepsilon} - \dot{\varepsilon}^p\right)\)

ii. Hardening variables#

\(\sigma_y\) = Yield stress

iii. Yield Surface#

\(f(\sigma,\sigma_y) = |\sigma| - \sigma_y\)

iv. Flow rule#

\(\dot{\varepsilon}^p = \gamma \cdot sign\left(\sigma\right)\)

v. Hardening Law#

\(\dot{\sigma_y} = 0\)

vi. Karush-Kuhn-Tucker complementary conditions#

\(\gamma \geq 0\)

\(f\left(\sigma, \sigma_y\right) \leq 0\)

\(\gamma \cdot f\left(\sigma, \sigma_y\right)=0\)

vii. Consistency condition#

\(\gamma \cdot \dot{f}\left(\sigma, \sigma_y\right) = 0\)

Return Mapping#

\(\sigma_{i}\) = stress value at start of current load step

\(\Delta \varepsilon\) = strain increment

\(\sigma_{i+1}^{trial} = \sigma_{i} + E \cdot \Delta \varepsilon\)

\(f_{i+1}^{trial} = \left|\sigma_{i+1}^{trial}\right| - \sigma_y\)

if \(f_{i+1}^{trial} < 0\):

\(\sigma_{i+1} = \sigma_{i+1}^{trial}\)

\(\varepsilon^p_{i+1} = \varepsilon^p_i\)

else:

\(\Delta \gamma = \frac{f_{i+1}^{trial}}{E}\)

\(\sigma_{i+1} = \sigma_{i+1}^{trial} - \Delta \gamma \cdot E \cdot sign \left(\sigma_{i+1}^{trial}\right)\)

\(\varepsilon^p_{i+1} = \varepsilon^p_i + \Delta \gamma \cdot sign \left(\sigma_{i+1}^{trial}\right)\)

%matplotlib notebook
import numpy as np
import matplotlib.pyplot as plt

E = 100
sigmay = 1
epsilon_y = sigmay/E
Npts = 1000 # Number of points per cycle in sinusoidal strain history
Ncyc = 3 # Number of cycles in sinusoidal strain history
N = Npts*Ncyc
time = np.arange(0,N)
epsilon_max = 5.0*epsilon_y
epsilon = epsilon_max*np.sin(2.0*np.pi*time/Npts)

# Now define stress
sigma = np.zeros(N)
for i in range(1,N):
    depsilon = epsilon[i] - epsilon[i-1]
    sigma_trial = sigma[i-1] + E*depsilon
    ftrial = np.abs(sigma_trial) - sigmay
    if(ftrial<0):
        sigma[i] = sigma_trial
    else:
        delta_gamma = ftrial/E
        sigma[i] = sigma_trial - delta_gamma*E*np.sign(sigma_trial)
plt.plot(epsilon,sigma)

Elastic With Kinematic Hardening#

Define Model#

i. Elastic stress-strain relationship#

\(\dot{\sigma} = E\left(\dot{\epsilon} - \dot{\epsilon}^p\right)\)

ii. Hardening variables#

\(\sigma_y\) = Yield stress

\(\sigma_q\) = Back stress

\(H\) = Kinematic hardening modulus

iii. Yield Surface#

\(f(\sigma,\sigma_y, \sigma_q) = |\sigma - \sigma_q| - \sigma_y\)

iv. Flow rule#

\(\dot{\epsilon}^p = \gamma \cdot sign\left(\sigma - \sigma_q\right)\)

v. Hardening Law#

\(\dot{\sigma_y} = 0\)

\(\dot{\sigma_q} = H \cdot \dot{\epsilon}^p\)

\(\dot{H} = 0\)

vi. Kuhn-Tucker complementary conditions#

\(\gamma \geq 0\)

\(f\left(\sigma, \sigma_y, q\right) \leq 0\)

\(\gamma \cdot f\left(\sigma, \sigma_y, \sigma_q\right)=0\)

vii. Consistency condition#

\(\gamma \cdot \dot{f}\left(\sigma, \sigma_y, \sigma_q\right) = 0\)

Return Mapping#

\(\sigma_{i}\) = stress value at start of current load step

\(\Delta \epsilon = \epsilon_{i+1} - \epsilon_i\) = strain increment

\(\sigma_{i+1}^{trial} = \sigma_{i} + E \cdot \Delta \epsilon\)

\(\sigma_{q,i+1}^{trial} = \sigma_{q,i}\)

\(f_{i+1}^{trial} = \left|\sigma_{i+1}^{trial} - \sigma_{q,i+1}^{trial}\right| - \sigma_y\)

if \(f_{i+1}^{trial} < 0\):

\(\sigma_{i+1} = \sigma_{i+1}^{trial}\)

\(\epsilon^p_{i+1} = \epsilon^p_i\)

\(\sigma_{q,i+1} = \sigma_{q,i+1}^{trial}\)

else:

\(\Delta \gamma = \frac{f_{i+1}^{trial}}{E + H}\)

\(\sigma_{i+1} = \sigma_{i+1}^{trial} - \Delta \gamma \cdot E \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i+1}^{trial}\right)\)

\(\epsilon^p_{i+1} = \epsilon^p_i + \Delta \gamma \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i+1}^{trial}\right)\)

\(\sigma_{q,i+1} = \sigma_{q,i} + \Delta \gamma \cdot H \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i+1}^{trial}\right)\)

%matplotlib notebook
import numpy as np
import matplotlib.pyplot as plt

E = 100000
H = 0.1*E
sigmay = 1000
epsilon_y = sigmay/E
Npts = 1000 # Number of points per cycle in sinusoidal strain history
Ncyc = 3 # Number of cycles in sinusoidal strain history
N = Npts*Ncyc
time = np.arange(0,N)
epsilon_max = 5.0*epsilon_y
epsilon = epsilon_max*np.sin(2.0*np.pi*time/Npts)

# Now define stress
sigma = np.zeros(N)
sigma_q = np.zeros(N)
epsilon_p = np.zeros(N)
for i in range(1,N):
    depsilon = epsilon[i] - epsilon[i-1]
    sigma_trial = sigma[i-1] + E*depsilon
    sigma_qtrial = sigma_q[i-1]
    ftrial = np.abs(sigma_trial-sigma_qtrial) - sigmay
    if(ftrial<0):
        sigma[i] = sigma_trial
        sigma_q[i] = sigma_qtrial
        epsilon_p[i] = epsilon_p[i-1]
    else:
        delta_gamma = ftrial/(E+H)
        sigma[i] = sigma_trial - delta_gamma*E*np.sign(sigma_trial-sigma_qtrial)
        sigma_q[i] = sigma_q[i-1] + delta_gamma*H*np.sign(sigma_trial-sigma_qtrial)
        epsilon_p[i] = epsilon_p[i-1] + delta_gamma*np.sign(sigma_trial-sigma_qtrial)
fig, ax = plt.subplots(2,2,sharex='all',sharey='all')
ax[0,0].plot(epsilon,sigma)
ax[0,1].plot(epsilon_p, sigma)
ax[1,0].plot(epsilon,sigma_q)
ax[1,1].plot(epsilon_p,sigma_q)
ax[1,0].set_xlabel(r'$\epsilon$')
ax[0,0].set_ylabel(r'$\sigma$')
ax[1,0].set_ylabel(r'$\sigma_q$')
ax[1,1].set_xlabel(r'$\epsilon_p$')
plt.tight_layout()

Elastic With Isotropic Hardening#

Consider a uniaxial spring that is linear elastic until yielding, and then is perfectly-plastic (i.e., zero post-yield stiffness).

Define Model#

i. Elastic stress-strain relationship#

\(\dot{\sigma} = E\left(\dot{\epsilon} - \dot{\epsilon}^p\right)\)

ii. Hardening variables#

\(\sigma_y\) = Yield stress

\(\alpha\) = function of the amount of plastic flow

\(K\) = Isotropic hardening modulus

iii. Yield Surface#

\(f(\sigma,\sigma_y, \alpha, K) = |\sigma| - \left[\sigma_y + K \alpha \right]\)

iv. Flow rule#

\(\dot{\epsilon}^p = \dot{\gamma} \cdot sign\left(\sigma\right)\)

v. Hardening Law#

\(\dot{\sigma_y} = 0\)

\(\dot{\alpha} = \left| \dot{\epsilon}^p \right|\)

\(\dot{K} = 0\)

vi. Kuhn-Tucker complementary conditions#

\(\gamma \geq 0\)

\(f\left(\sigma, \sigma_y, \alpha, K \right) \leq 0\)

\(\gamma \cdot f\left(\sigma, \sigma_y, \alpha, K \right)=0\)

vii. Consistency condition#

\(\gamma \cdot \dot{f}\left(\sigma, \sigma_y, \alpha, K \right) = 0\)

Return Mapping#

\(\sigma_{i}\) = stress value at start of current load step

\(d\epsilon\) = strain increment

\(\sigma_{i+1}^{trial} = \sigma_{i} + E \cdot d\epsilon\)

\(f\left(\sigma_{i+1}^{trial}, \sigma_y, \alpha, K \right) = \left|\sigma_{i+1}^{trial}\right| - \left[ \sigma_y + K \alpha \right]\)

if \(f\left(\sigma_{i+1}^{trial}, \sigma_y, \alpha, K \right) < 0\):

\(\sigma_{i+1} = \sigma_{i+1}^{trial}\)

\(\epsilon^p_{i+1} = \epsilon^p_i\)

\(\alpha_{i+1} = \alpha_i\)

else:

\(\Delta \gamma = \frac{f\left(\sigma_{i+1}^{trial},\sigma_y, \alpha, K \right)}{E + K}\)

\(\sigma_{i+1} = \sigma_{i+1}^{trial} - \Delta \gamma \cdot E \cdot sign \left(\sigma_{i+1}^{trial}\right)\)

\(\epsilon^p_{i+1} = \epsilon^p_i + \Delta \gamma \cdot sign \left(\sigma_{i+1}^{trial}\right)\)

\(\alpha_{i+1} = \alpha_i + \Delta \gamma\)

%matplotlib notebook
import numpy as np
import matplotlib.pyplot as plt

E = 100000
K = 10000
sigmay = 1000
epsilon_y = sigmay/E
Npts = 1000 # Number of points per cycle in sinusoidal strain history
Ncyc = 5 # Number of cycles in sinusoidal strain history
N = Npts*Ncyc
time = np.arange(0,N)
epsilon_max = 5.0*epsilon_y
epsilon = []
alpha = []
for i in range(N):
    epsilon.append(epsilon_max*np.sin(2.0*np.pi*i/Npts))

# Now define stress
sigma = np.zeros(N)
alpha = np.zeros(N)
epsilon_p = np.zeros(N)
alpha[0] = 0.0
epsilon_p[0] = 0.0
for i in range(1,N):
    depsilon = epsilon[i] - epsilon[i-1]
    sigma_trial = sigma[i-1] + E*depsilon
    ftrial = np.abs(sigma_trial) - (sigmay+K*alpha[i-1])
    if(ftrial<0):
        sigma[i] = sigma_trial
        alpha[i] = alpha[i-1]
        epsilon_p[i] = epsilon_p[i-1]
    else:
        delta_gamma = ftrial/(E+K)
        sigma[i] = sigma_trial - delta_gamma*E*np.sign(sigma_trial)
        alpha[i] = alpha[i-1] + delta_gamma
        epsilon_p[i] = epsilon_p[i-1] + delta_gamma*np.sign(sigma_trial-K*alpha[i-1])
fig, ax = plt.subplots(nrows=2,ncols=2,sharex='all',sharey='row')
ax[0,0].plot(epsilon,sigma)
ax[0,1].plot(epsilon_p, sigma)
ax[1,0].plot(epsilon,alpha)
ax[1,1].plot(epsilon_p,alpha)
ax[1,0].set_xlabel(r'$\epsilon$')
ax[0,0].set_ylabel(r'$\sigma$')
ax[1,0].set_ylabel(r'$\alpha$')
ax[1,1].set_xlabel(r'$\epsilon_p$')
plt.tight_layout()

Elastic With Kinematic and Isotropic Hardening#

Consider a uniaxial spring that is linear elastic until yielding, and then is perfectly-plastic (i.e., zero post-yield stiffness).

Define Model#

i. Elastic stress-strain relationship#

\(\dot{\sigma} = E\left(\dot{\epsilon} - \dot{\epsilon}^p\right)\)

ii. Hardening variables#

\(\sigma_y\) = Yield stress

\(\sigma_q\) = Back stress

\(H\) = Kinematic hardening modulus

\(\alpha\) = Function of the amount of plastic flow

\(K\) = Isotropic hardening modulus

iii. Yield Surface#

\(f(\sigma,\sigma_y, \sigma_q, \alpha, K) = |\sigma - \sigma_q| - \left[\sigma_y + K \alpha \right]\)

iv. Flow rule#

\(\dot{\epsilon}^p = \dot{\gamma} \cdot sign\left(\sigma - \sigma_q \right)\)

v. Hardening Law#

\(\dot{\sigma_y} = 0\)

\(\dot{K} = 0\)

\(\dot{H} = 0\)

\(\dot{\alpha} = \left| \dot{\epsilon}^p \right|\)

\(\dot{\sigma_q} = H \cdot \dot{epsilon}^p\)

vi. Kuhn-Tucker complementary conditions#

\(\gamma \geq 0\)

\(f\left(\sigma, \sigma_y, \sigma_q, \alpha, K \right) \leq 0\)

\(\gamma \cdot f\left(\sigma, \sigma_y, \sigma_q, \alpha, K \right)=0\)

vii. Consistency condition#

\(\gamma \cdot \dot{f}\left(\sigma, \sigma_y, \sigma_q, \alpha, K \right) = 0\)

Return Mapping#

\(\sigma_{i}\) = stress value at start of current load step

\(d\epsilon\) = strain increment

\(\sigma_{i+1}^{trial} = \sigma_{i} + E \cdot d\epsilon\)

\(f\left(\sigma_{i+1}^{trial}, \sigma_y, \sigma_{q,i}, \alpha_i, K \right) = \left|\sigma_{i+1}^{trial} - \sigma_{q,i}\right| - \left[ \sigma_y + K \alpha_i \right]\)

if \(f\left(\sigma_{i+1}^{trial}, \sigma_y, \sigma_{q,i}, \alpha_i, K \right) < 0\):

\(\sigma_{i+1} = \sigma_{i+1}^{trial}\)

\(\epsilon^p_{i+1} = \epsilon^p_i\)

\(\alpha_{i+1} = \alpha_i\)

\(\sigma_{q,i+1} = \sigma_{q,i}\)

else:

\(\Delta \gamma = \frac{f\left(\sigma_{i+1}^{trial},\sigma_y, \sigma_{q,i}, \alpha, K \right)}{E + K + H}\)

\(\sigma_{i+1} = \sigma_{i+1}^{trial} - \Delta \gamma \cdot E \cdot sign \left(\sigma_{i+1}^{trial} - \sigma_{q,i}\right)\)

\(\epsilon^p_{i+1} = \epsilon^p_i + \Delta \gamma \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i}\right)\)

\(\alpha_{i+1} = \alpha_i + \Delta \gamma\)

\(\sigma_{q,i+1} = \sigma_{q,i} + \Delta \gamma \cdot H \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i}\right)\)

%matplotlib notebook
import numpy as np
import matplotlib.pyplot as plt

E = 100000
K = 10000
H = 10000
sigmay = 1000
epsilon_y = sigmay/E
Npts = 1000 # Number of points per cycle in sinusoidal strain history
Ncyc = 5 # Number of cycles in sinusoidal strain history
N = Npts*Ncyc
time = np.arange(0,N)
epsilon_max = 5.0*epsilon_y
epsilon = []
for i in range(N):
    epsilon.append(epsilon_max*np.sin(2.0*np.pi*i/Npts))

# Now define stress
sigma = np.zeros(N)
alpha = 0.0
q=0.0
for i in range(1,N):
    depsilon = epsilon[i] - epsilon[i-1]
    sigma_trial = sigma[i-1] + E*depsilon
    ftrial = np.abs(sigma_trial-q) - (sigmay+K*alpha)
    if(ftrial<0):
        sigma[i] = sigma_trial
    else:
        delta_gamma = ftrial/(E+K+H)
        sigma[i] = sigma_trial - delta_gamma*E*np.sign(sigma_trial-q)
        alpha = alpha + delta_gamma
        q = q + delta_gamma*H*np.sign(sigma_trial-q)
plt.plot(epsilon,sigma)
