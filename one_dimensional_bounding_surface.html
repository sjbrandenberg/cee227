Bounding Surface Plasticity with Kinematic Hardening#

Define Model#

i. Elastic stress-strain relationship#

\(\dot{\sigma} = E\left(\dot{\epsilon} - \dot{\epsilon}^p\right)\)

ii. Hardening variables#

\(\sigma_y\) = Yield stress

\(\sigma_{max}\) = Bounding surface stress

\(\sigma_q\) = Back stress

\(H\) = Kinematic hardening modulus

iii. Yield Surface#

\(f(\sigma,\sigma_y, \sigma_q) = |\sigma - \sigma_q| - \sigma_y\)

iv. Flow rule#

\(\dot{\epsilon}^p = \gamma \cdot sign\left(\sigma - \sigma_q \right)\)

v. Hardening Law#

\(\dot{\sigma_y} = 0\)

\(\dot{H} = 0\)

\(\dot{\sigma_q} = H \cdot \frac{\left| \sigma_{max} \cdot sign(\sigma - \sigma_q) - \sigma_y \right|}{\left | \sigma - \sigma_y \cdot sign(\sigma - \sigma_q) \right|} \dot{\epsilon}^p\)

vi. Kuhn-Tucker complementary conditions#

\(\gamma \geq 0\)

\(f\left(\sigma, \sigma_y, \sigma_q \right) \leq 0\)

\(\gamma \cdot f\left(\sigma, \sigma_y, \sigma_q, H \right)=0\)

vii. Consistency condition#

\(\gamma \cdot \dot{f}\left(\sigma, \sigma_y, \sigma_q, \right) = 0\)

Return Mapping#

\(\sigma_{i}\) = stress value at start of current load step

\(d\epsilon\) = strain increment

\(\sigma_{i+1}^{trial} = \sigma_{i} + E \cdot d\epsilon\)

\(f\left(\sigma_{i+1}^{trial}, \sigma_y, \sigma_{q,i}, \alpha_i, K \right) = \left|\sigma_{i+1}^{trial} - \sigma_{q,i}\right| - \left[ \sigma_y + K \alpha_i \right]\)

if \(f\left(\sigma_{i+1}^{trial}, \sigma_y, \sigma_{q,i}, \alpha_i, K \right) < 0\):

\(\sigma_{i+1} = \sigma_{i+1}^{trial}\)

\(\epsilon^p_{i+1} = \epsilon^p_i\)

\(\alpha_{i+1} = \alpha_i\)

\(\sigma_{q,i+1} = \sigma_{q,i}\)

else:

\(\Delta \gamma = \frac{f\left(\sigma_{i+1}^{trial},\sigma_y, \sigma_{q,i}, \alpha, K \right)}{E + K + H}\)

\(\sigma_{i+1} = \sigma_{i+1}^{trial} - \Delta \gamma \cdot E \cdot sign \left(\sigma_{i+1}^{trial} - \sigma_{q,i}\right)\)

\(\epsilon^p_{i+1} = \epsilon^p_i + \Delta \gamma \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i}\right)\)

\(\alpha_{i+1} = \alpha_i + \Delta \gamma\)

\(\sigma_{q,i+1} = \sigma_{q,i} + \Delta \gamma \cdot H \cdot sign \left(\sigma_{i+1}^{trial}-\sigma_{q,i}\right)\)

%matplotlib notebook
import numpy as np
import matplotlib.pyplot as plt

sigma_y = 0.1
sigma_max = 2
E = 100
H = 50
Npts = 100 # Number of points per cycle in sinusoidal strain history
Ncyc = 10 # Number of cycles in sinusoidal strain history
N = Npts*Ncyc
time = np.arange(0,N)
epsilon_y = sigma_y/E
epsilon_max = 50.0*epsilon_y
epsilon = []
for i in range(N):
    epsilon.append(epsilon_max*np.sin(2.0*np.pi*i/Npts))

sigma = np.zeros(N)
sigma_q = np.zeros(N)
epsilon_p = np.zeros(N)
I = 0
for i in range(1,N):
    depsilon = epsilon[i] - epsilon[i-1]
    sigma_trial = sigma[i-1] + E*depsilon
    sigma_qtrial = sigma_q[i-1]
    ftrial = np.abs(sigma_trial - sigma_qtrial) - sigma_y
    if(ftrial<0):
        I = 0
        sigma[i] = sigma_trial
        sigma_q[i] = sigma_qtrial
        epsilon_p[i] = epsilon_p[i-1]
    else:
        SIGN = np.sign(sigma_trial - sigma_qtrial)
        if(I==0):
            sigma_in = sigma_y*SIGN+sigma_q[i-1]
            I = 1
        a = -E*E*SIGN+E*H*SIGN
        b = sigma_trial*E - sigma_in*E + H*sigma_max*SIGN - H*sigma_trial + E*SIGN*ftrial
        c = -sigma_trial*ftrial + sigma_in*ftrial
        delta_gamma = (-b+SIGN*np.sqrt(b*b-4*a*c))/(2*a)
        sigma[i] = sigma_trial - delta_gamma*E*SIGN
        sigma_q[i] = sigma_q[i-1] + delta_gamma*H*((sigma_max*SIGN-sigma[i])/(sigma[i]-sigma_in))*SIGN
        epsilon_p[i] = epsilon_p[i-1] + delta_gamma*SIGN
fig, ax = plt.subplots(2,2,sharex='all',sharey='all')
ax[0,0].plot(epsilon,sigma)
ax[0,1].plot(epsilon_p, sigma)
ax[1,0].plot(epsilon,sigma_q)
ax[1,1].plot(epsilon_p,sigma_q)
ax[1,0].set_xlabel(r'$\epsilon$')
ax[0,0].set_ylabel(r'$\sigma$')
ax[1,0].set_ylabel(r'$\sigma_q$')
ax[1,1].set_xlabel(r'$\epsilon_p$')
plt.tight_layout()
